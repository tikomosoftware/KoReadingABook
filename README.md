# KoReadingABook v1.0.1 📖
**Windows Active Window Simulator**

![KoReadingABook](https://img.shields.io/badge/version-0.1.0--alpha-orange)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![.NET](https://img.shields.io/badge/.NET-10.0-purple)

KoReadingABook は、PCの操作をシミュレートするユーティリティツールです。
定期的に画面上のウィンドウをランダムにアクティブ化（最前面へ移動）し、マウスカーソルを自動的に動かすことで、ユーザーがPCを操作しているかのような状態を維持します。

## 主な機能 🚀

### 1. ランダムウィンドウアクティブ化
- 指定した間隔（デフォルト15秒）で、現在開いているアプリの中からランダムに1つ選んで最前面に表示します。
- **スマートな切り替え**:
  - 現在最前面にあるアプリとは**異なる**アプリを必ず選択します。
  - 「設定」や「タスクマネージャー」、本アプリ自身は選択対象から除外されます。
  - 最小化されているアプリも復元して表示します。

### 2. マウス自動円運動
- 1秒ごとにマウスカーソルが画面中央付近でゆっくりと円を描くように動きます。
- **ユーザー操作の優先**:
  - マウスを自分で動かすと、自動移動機能が**10秒間一時停止**します。
  - 10秒間操作がない場合、自動的に円運動が再開されます。
  - これにより、ツールを停止することなく、急な作業や停止ボタンのクリックがスムーズに行えます。

## 使い方 💡

1. **アプリを起動**: `KoReadingABook.exe` を実行します。
2. **待機状態**: 起動直後は「Ready」状態で待機しており、何も動作しません。ウィンドウは画面右下に配置されます。
3. **設定**: 必要に応じて左下のプルダウンからウィンドウ切り替え間隔（10秒〜30秒）を選択します。
4. **開始**: 緑色の **[Start]** ボタンをクリックすると動作を開始します。
5. **停止**: 赤色の **[Stop]** ボタンをクリックすると全ての動作が停止します。

## 動作環境 💻

- OS: Windows 10 / 11
- 必須ランタイム: .NET 10.0 Desktop Runtime

## 注意事項 ⚠️

- 本ツールは、アクティブなウィンドウを強制的に切り替えるため、**文字入力中やゲーム中などの使用は推奨されません**。
- 一部の管理者権限で動作するアプリケーション（タスクマネージャー等）に対しては、本ツールも管理者として実行しない限り制御できない場合があります。
- マウス移動中はスクリーンセーバーが起動しない場合があります。
